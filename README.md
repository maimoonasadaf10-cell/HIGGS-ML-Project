# HIGGS Boson Classification using Machine Learning

## Project Overview
This project classifies particle collision events as signal (Higgs Boson) or background using machine learning.

## Dataset
HIGGS Dataset from UCI Machine Learning Repository

## Algorithms Used
- Logistic Regression
- Random Forest
- Linear SVM

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Tools
- Python
- Google Colab
- Scikit-Learn
- Pandas
- Matplotlib
- Seaborn
